TROUBLESHOOTING
═══════════════════════════════════════════════════════════════

Problem: Console still shows "100 cents"
Solution:
  • Verify files uploaded to correct location
  • Clear browser cache completely
  • Try incognito window
  • Check file timestamps on server

Problem: "Card declined" error with test card
Solution:
  • Test cards only work in Square Sandbox
  • Verify using correct test card: 4111 1111 1111 1111
  • If using production credentials, use real card (BE CAREFUL!)

Problem: Square charges wrong amount
Solution:
  • Check backend deployed correctly
  • Verify BOOKING_PRICE_CENTS = 32292 in Apps Script
  • Check frontend config.js has BOOKING_PRICE: 322.92

Problem: Email doesn't show pricing breakdown
Solution:
  • Check bookNow.js uploaded correctly
  • Verify it includes lines 2818-2822 with pricing fields
  • Clear browser cache and retry
